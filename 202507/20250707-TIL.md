# 컬렉션 프레임워크
## 1. List
### - 순서가 있는 데이터구조
### - 중복데이터 저장을 허용
### - 종류: 배열리스트, 연결리스트
## 2. Map
### - JSON 객체, 파이썬 딕셔너리 대응
### - key와 value를 가진 엔트리 객체를 보관하는 자료구조
### - key는 중복을 허용하지 않으며 value는 중복을 허용
### - 순서를 보장하지 않음
## 3. Set
### - 집합
### - 중복을 허용하지 않음
### - 순서를 보장하지 않음 (해시)
