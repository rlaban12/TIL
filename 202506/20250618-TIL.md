## 2. 서버에서 데이터 가져오기
### - 데이터 요청 시 사용하는 4가지
#### 1) GET : 조회
#### 2) POST : 생성
#### 3) PUT / PATCH : 수정
#### 4) DELETE : 삭제
### - xhr 방식으로 서버에 요청하게 되면 callback-hell에 빠지게 된다.
#### - 1) 이유: xhr는 비동기 방식으로 동시에 순서없이 실행되기 때문에 순서를 맞춰주기 위해서 callback을 쓰게 되기 때문이다.
